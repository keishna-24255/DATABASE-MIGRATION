# Database Migration (MySQL to PostgreSQL)

## About the Project
This project demonstrates database migration from MySQL to PostgreSQL using Python.
It ensures data integrity while transferring records between databases.

## Objective
- Migrate data between two databases
- Maintain data accuracy and integrity
- Learn real-world database handling

## Technologies Used
- Python
- MySQL
- PostgreSQL
- mysql-connector-python
- psycopg2

## How It Works
1. Connects to MySQL database
2. Fetches records from source table
3. Creates table in PostgreSQL
4. Inserts records into PostgreSQL
5. Displays migrated data
